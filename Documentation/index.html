Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside Sparrk folder.

Pages

    • index.html is for Default Home page
    • index-2.html is for Default Home-2 page
    • index-animatetext.html is for animate text slider page
    • index-boxed.html is for Box layout Home page
    • index-dotted-overlay.html is for Dotted Overlay Home page
    • index-parallax.html is for Parallex Home page
    • index-video.html is for Background Video Home page
    • index-youtube-video.html is for Youtube Video Home page
    • blog.html is for blog page
    • blog-leftsidebar.html is for blog left sidebar page
    • event-details.html is for event details page
    • event-schedule.html is for event shedule page
    • single-blog.html is for Single Blog page
    • single-blog-leftsidebar.html is for single blog leftsidebar page
    • 404.html is for single blog leftsidebar page
